# AEGIS Final Presentation Script

## 1. Introduction
AEGIS is an autonomous agentic framework for incident response and root cause analysis in cloud systems.

## 2. Problem
Manual RCA is slow because SREs must inspect logs, metrics, alerts, historical incidents, and runbooks manually.

## 3. Proposed Solution
AEGIS combines RAG, OpenAI, multi-agent reasoning, graph RCA, GNN-style ranking, and human-approved remediation.

## 4. Demo Flow
1. Select an incident.
2. Run end-to-end AEGIS analysis.
3. Explain root cause and confidence.
4. Show evidence and citations.
5. Show multi-agent timeline.
6. Show graph RCA candidates.
7. Show GNN-style RCA Top-3.
8. Show remediation plan.
9. Approve a safe remediation action.
10. Show approval audit log and MTTR dashboard.

## 5. Closing
AEGIS improves RCA explainability, reduces simulated MTTR, and supports safe autonomous remediation with human approval.
